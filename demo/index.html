scroll-back-button

Does not provide any style and structure, you need to handwrite the structure in scroll-back-button. Specify the direction and parent attributes to start scrolling!

Code snippets

Back to Top

Feel it 
            
<scroll-back-button>
  <button>Feel it</button>
</scroll-back-button>
          
        
            
<scroll-back-button direction="top">
  <button>Feel it</button>
</scroll-back-button>

Back to Bottom

Feel it 
            
<scroll-back-button direction="bottom">
  <button>Feel it</button>
</scroll-back-button>
            
          
            
<scroll-back-button direction="bot">
  <button>Feel it</button>
</scroll-back-button>

Scroll in box

Feel it(TOP) Feel it(BOTTOM) 
            
<div style="height: 100px; overflow: auto;">
  <div id="boxOne" style="height: 2000px;">
  </div>
</div>
            
          
            
<scroll-back-button parent="#boxOne">
  <button>Feel it(TOP)</button>
</scroll-back-button>
            
          
            
<scroll-back-button parent="#boxOne" direction="bot">
  <button>Feel it(BOTTOM)</button>
</scroll-back-button>
arrow_upward Scroll to TOP
arrow_downward Scroll to BOTTOM
